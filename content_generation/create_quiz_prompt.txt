For the NES video game Legend of Zelda, please create a quiz. The quiz should have 10 easy difficulty questions followed by 10 medium difficulty questions followed by 10 hard difficulty questions. The answers to each question should be multiple choice. There should be 4 answers per question and only one of the answers should be correct.

Once the quiz has been created, turn it into JSON with the following structure.

{
    "videoGameName": "",
    "questions": [{
        "questionText": "",
        "difficulty": "",
        "answers": [{
            "answerText": "",
            "isCorrectAnswer": ""
        }]
    }]
}
